Data type: PronounceableText.
PropertyExpected TypeDescription
Properties from PronounceableText
inLanguage Language  or
Text		 The language of the content or performance or used in an action. Please use one of the language codes from the IETF BCP 47 standard. See also availableLanguage. Supersedes language.
phoneticText Text Representation of a text textValue using the specified speechToTextMarkup. For example the city name of Houston in IPA: /ˈhjuːstən/.
speechToTextMarkup Text Form of markup used. eg. SSML or IPA.
textValue Text Text value being annotated.
