Schema.org Section: bib

Schema.org is a set of extensible schemas that enables webmasters to embed structured data on their web pages for use by search engines and other applications. For more details, see the homepage.

The Bibliographic Section within schema.org defines terms such as Audiobook, Thesis, ComicStory, and workTranslation. For more details see the W3C BibExtend Community Group's wiki.
